# Semantic Search Design

## Purpose

This document explains how curated insurance claim records can be prepared for semantic search and retrieval-augmented generation patterns.

The goal is to demonstrate how a Data Engineer can convert structured Gold/PIT data into search-ready documents that downstream AI tools can use for claim discovery, summarization, and natural language question answering.
