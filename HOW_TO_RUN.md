# How to Run This Project Locally

This guide explains how to run the Fabric AI Claims Intelligence Platform locally using Python and CSV files.

The project simulates a Microsoft Fabric-style insurance claims lakehouse with Bronze, Silver, Gold, Point-in-Time, and AI-ready output layers.

## Prerequisites

Before running the project, make sure you have:

- Python installed
- Git installed, optional if using GitHub Desktop
- VS Code or any code editor
- Basic terminal access

This project uses `pandas` for local data processing.

## 1. Clone the Repository

```bash
git clone https://github.com/etl-kenobi/fabric-ai-claims-intelligence-platform.git
cd fabric-ai-claims-intelligence-platform
